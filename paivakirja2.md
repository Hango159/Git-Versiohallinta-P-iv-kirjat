# Oppimispäiväkirja: Hajautettu git

**Mikä osion tehtävissä oli vaikeaa ja mikä helppoa? Mikä auttoi minua oppimaan? Miten selvitin esteet, jotka vaikuttivat tehtävän suorittamiseen?**

Harjoituksen haasteet ja onnistumiset: Erityisen opettavaista oli nähdä, miten paikallinen repositorio ja GitHubin etärepositorio keskustelevat keskenään. Aluksi git fetch -komennon ja haaran yhdistämisen välinen ero tuntui hieman sekavalta, mutta kun näin tiedostojen ilmestyvän levylle vasta yhdistämisen jälkeen, asia selkeytyi.

## Osiossa käyttämäni Git-komennot

| Komento      | Kuvaus                                                    |
| ------------ | --------------------------------------------------------- |
| git remote   | Näet etärepositoriot ja niiden asetukset                  |
| git fetch    | Hakee muutokset epärepositoriosta, mutta ei yhdistä niitä |
| git pull     | Yhdistää muutokset paikalliseen repositorioon             |
| git checkout | Vaihtaa branchin ja muokkaa tiedostoja hakemistossasi     |
