# **Mikä osion tehtävissä oli vaikeaa ja mikä helppoa? Mikä auttoi minua oppimaan? Miten selvitin esteet?**

Tehtävät olivat pääosin helppoja, ja pystyin tekemään ne ilman suurempia vaikeuksia. Selkeät ohjeet tekivät etenemisestä sujuvaa.

Jos jokin tuntui aluksi hankalalta, katsoin ohjeet uudelleen ja etsin tarvittavat tiedot kurssimateriaaleista. Niistä löytyi kaikki tarvittava.

## Osiossa käyttämäni Git-komennot

| Komento      | Kuvaus                                                   |
| ------------ | -------------------------------------------------------- |
| git init     | komennolla perustetaan paikallinen repositorio           |
| git status   | komennolla näkee hakemiston ja sen muutosten Git-tilan   |
| git add      | komennolla voit lisätä talletukseen tiedostoja           |
| git commit   | tallennus tehdään tällä komennolla                       |
| git log      | talletusten historian näkee tällä komennolla             |
| git rm       | poistaa tiedoston Git-hallinasta, sekä työhakemistosta   |
| git mv       | voi nimetä tiedoston uudelleen, tai siirtää sitä         |
| git reset    | peruuttaa tiedoston lisäyksen tallenuksesta              |
| git restore  | peruuttaa työtilassa tekemiä muutoksia,                  |
| -----------  | joita ei olla vielä talletettu versionhallintaan         |
| git branch   | voidaan luoda uusia haaroja ja katsoa olemassa olevat    |
| ----------   | haarat                                                   |
| git switch   | komennolla vaihdetaan haaraa                             |
| git merge    | tekee uuden tallennuksen, jossa on yhdistetty molempien  |
| ---------    | haarojen muutokset                                       |
| git checkout | pääset näkymään, jossa voit vaihtaa haaraa tai palauttaa |
| ---------    | työhakemiston tiedostoja.                                |
